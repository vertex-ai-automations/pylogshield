Which features are must-haves?

Select everything you want in the TUI. We'll scope the design around your priorities.

1

Live Search Bar

Type to filter — results update as you type. Supports plain text and regex. Highlights matches inline.

2

Filter Panel

Sidebar with toggles for log level (DEBUG/INFO/WARNING/ERROR/CRITICAL), time range (last 1h, 24h, custom), and logger name. Multiple filters stack.

3

Stats & Summary Bar

A header bar showing counts per level (e.g. 3 CRITICAL · 12 ERROR · 847 INFO), error rate, and time span of visible logs.

4

Export

Export the current filtered view to CSV, JSON, or plain text. Useful for sharing a subset of logs with a team or attaching to a ticket.

5

Detail Panel

Press Enter on any row to expand a full detail view — complete JSON fields, stack trace, all context fields — without leaving the TUI.

6

Live Follow Toggle

A keybind (e.g. F) to switch between static view and live-follow mode. A visible indicator shows which mode is active. Auto-pauses when you scroll up.

7

Bookmarks

Mark rows of interest with B and jump between them. Useful during incident investigation to tag key events before exporting.

8

Keyboard Help Overlay

Press ? to show all keybindings in a modal. Essential for discoverability in a keyboard-driven TUI.
